# FIFA World Cup 2026 Landing Page

A modern and responsive landing page inspired by the FIFA World Cup 2026 theme. The website showcases tournament highlights, host cities, featured matches, and a polished user experience built with HTML and CSS.

## Features

- Responsive layout for desktop, tablet, and mobile screens
- Hero section with bold tournament branding
- Featured host city cards
- Match schedule table
- Clean and modern visual styling
- Optimized for fast and simple browsing

## Project Structure

- index.html - Main structure of the landing page
- style.css - All visual styling and responsive behavior
- assets/images - Images used across the site

## How to View the Project

1. Open the project folder in your browser.
2. Double-click index.html, or use a local development server.

Example with VS Code Live Server:

- Install the Live Server extension
- Right-click index.html
- Select "Open with Live Server"
